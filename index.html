Secure intra-vehicular communication over Central Gateway in Connected Vehicle System

The Connected Vehicles (CV) are such systems that are formulated with hardware and software systems having vulnerabilities. When any part of a vehicle tries to communicate with any other part of the same vehicle, the security is at risk as an exchange of information takes place. This communication between in-vehicle parts is done through a Gateway. With the intension to have keep it safe so that no security breach takes place, in project paper I have designed a mechanism using cryptography and MAC to provide security to the gateway such that the in-vehicular communication is perfectly secure. Provided the attack scenarios and counter measures to ensure foolproof and reliable security system.
